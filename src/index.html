CoderDojo Tokyo

CoderDojoは小中学生のためのプログラミング道場

CoderDojo Tokyoでは、毎週日曜日、ボランティアによるプログラミング道場を開催しています。

  • Scratch (ブロックを組合わせて簡単プログラミング)
  • HTML / CSS / JavaScript
  • YouTubeなどを活用したWEBサイト作成
  • 電子工作と組合わせたプログラミング
